# TopicRadar 版本號

**當前版本**: v2.1.0

## 更新版本號步驟

每次發布新版本時：

1. 更新 `CHANGELOG.md`（添加版本記錄）
2. 更新 `index.html` 底部版本顯示（第 61 行）
3. 提交時 commit 訊息包含版本號

## 版本號規則

遵循語意化版本號 (Semantic Versioning 2.0.0)

**格式**: `主版本號.次版本號.修訂號`

### 主版本號 (Major)
增加條件：
- 重大架構變更
- 不向後相容的 API 改動
- 重大功能重寫

範例：v1.x → v2.x（引入認證系統）

### 次版本號 (Minor)
增加條件：
- 新增功能
- 向後相容的改進
- 新增 API 端點

範例：v2.0 → v2.1（多語言關鍵字）

### 修訂號 (Patch)
增加條件：
- 錯誤修復
- 小幅優化
- 安全性修補
- 文件更新（僅文件）

範例：v2.1.0 → v2.1.1
